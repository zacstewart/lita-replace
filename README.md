# lita-replace

Fixes your stupid typos with VIM-like find and replace.

## Installation

Add lita-replace to your Lita instance's Gemfile:

```ruby
gem "lita-replace"
```

## Usage

zacstewart: whoops, I did that
zacstewart: s/whoops/awesome
Lita: zacstewart: awesome, I did that
